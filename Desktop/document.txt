  -> roslaunch rplidar_ros rplidar.launch #running rplidar
running rplidar with rviz visualization
  -> roslaunch rplidar_ros view_rplidar.launch
running hector slam
  -> roslaunch hector_slam_launch tutorial.launch

  -> roscore
  -> rosrun rosserial_python serial_node.py _port:=/dev/ttyACM0 _baud:=57600

      -> rostopic echo /package

robot motor stop command
  -> rostopic pub driving std_msgs/String error --once
  or
  -> rostopic pub driving std_msgs/String motor_stop --once

cartographer launch
  -> roslaunch gbot_core gbot.launch
  -> roslaunch gbot_core visualization.launch

error report
  1. python3 matplotlib install error
    -> sudo apt-get install python3-matplotlib
  2. python input error name is not defined
    -> please use raw_input() instead of input() at python2.x
  3. opencv-python install error
    -> sudo apt -y install python-opencv
